# CharmPlayer（魅听）
--------------------

## 简介
------

一个基于本地音乐的播放器。将音乐文件预先扫描存储生成播放列表并供用户播放。
目前项目还在开发阶段，预计包含后台播放、定时停止/退出、音效调节等功能及便捷的用户交互策略。
